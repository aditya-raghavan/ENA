Solar PV Panel Diode Junction Box Test Rig

This machine is for testing the Diode junction box used in Photovoltaic solar panels with upto 3 Diodes The test rig tests the diodes for both blocking voltage individually by applying reverse voltage and also the bypass operation during shadow conditions by applying rated current to the junction box. The machine is capable of applying currents upto 40 Amps and reverse voltage upto 48 Volts. More variations in Current and voltages are possible on request. This is an automatic PC based test rig. The test parameters are stored in the PC for easy retrieval. The test results are stored in the PC for TRAECEBILITY along with date/time stamp with serial number and batch information. The machine has a Bar code printer that can print barcode labels for stcking on the product. A bar code scanner is also part of the machine.

Grinding Wheel Dressing Machine

  • Grinding Wheel Dressing Machine - Projector Based

This machine is for Dressing of the grinding wheels used in tool and cutter manufacturing industry. The machine has a telecentric lens mounted with an optical profile projector. The machine is capable of dressing any type of Diamond / CBN / Vitrified wheels

  • Grinding Wheel Dressing Machine - PC Vision system Based

This machine is for Dressing of the grinding wheels used in tool and cutter manufacturing industry. The machine has a variable zoom telecentric lens mounted with an vision camera. A computer system is provided with a 19 inch monitor that projects the camera image overlayed with the CAD drawing of the dressing profile. The computer is preloaded with the vision inspection software for reading and overlaying the CAD files of the wheel profiles. Additional attachements such a linear scales and rotary encoders for the axis are provided based on customer requirements. The machine is capable of dressing any type of Diamond / CBN / Vitrified wheels
